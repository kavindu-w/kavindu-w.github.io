Evolution of Cooperation in LLM-Agent Societies: A Preliminary Study Using Different Punishment Strategies

● Final year research project studying normative reasoning abilities of LLM agents in social dilemmas using the Smallville multi-agent simulation environment, under the supervision of Prof. Stephen Cranefield (Otago, NZ), Prof. Tony Savarimuthu (Otago, NZ), Dr. Surangika Ranathunga (Massey, NZ), and Dr. Nisansa de Silva (UoM, SL)

● Modified the Smallville environment to model and simulate the diner's dilemma scenario, including implementing the dilemma scenario logic, agent scheduling, developing and fine-tuning the logic for punishment mechanisms, implementing detailed dilemma decisions and reasoning logging

● Lead the paper writing by drafting the abstract, literature review, and conclusion.

● Accepted and presented at the International COINE Workshop (C-ranked), co-located with AAMAS 2025, Detroit, Michigan, USA — 20th May 2025

Technologies Used: Python, Tiled Map Editor, Vertex API, Groq API

LangTrend — Tracking Language Representation in NLP Research

● Weekly automated pipeline that scans arXiv cs.CL submissions, detects explicitly mentioned human languages via full-text extraction (HTML → PDF → abstract fallback), and publishes an interactive dashboard tracking language representation over time

● Implements 6-tier language resource classification (Joshi et al. / Ranathunga & de Silva taxonomy), acronym conflict detection, and transparent false-positive flagging with links to source papers for manual verification

● GitHub Actions workflow runs every Tuesday; all manifest JSONs are committed to the repository as open data

Technologies Used: Python, arXiv API, Docling, Astro
